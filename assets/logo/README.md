# Logo Placement

Please place your logo file here with the name: `logo.jpg`

The logo should be:
- Named exactly: `logo.jpg`
- Recommended size: 512x512 pixels or higher
- Format: JPG or JPEG
- Square aspect ratio works best

The splash screen will automatically use this logo file when it's placed in this directory.

If the logo.jpg file is not found, the splash screen will display a fallback shopping bag icon instead.
